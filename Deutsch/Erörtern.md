# Arten der Erörterung
## Lineare Erörterung

#### Aufbau
- **Einleitung**
  - Interessante Hinführung und These
- **Hauptteil**
  - These
  - 1. Argument und Erklärung (Beispiel)
  - 2. Argument und Erklärung (Beispiel) – stärker, umfangreicher darstellen
  - 3. Argument und Erklärung + Beispiel – noch stärker erörtern
  - ...
- **Fazit**
  - Schluss – interessanter Ausstieg

## Dialektische Erörterung

#### Aufbau (Sanduhrprinzip)
- **Einleitung**
  - Interessante Hinführung
  - Deutlich machen, dass es 2 Meinungen gibt
- **Hauptteil**
  - Formulierung der Gegenthese
  - 1. Gegenargument (stärkstes)
  - 2. Gegenargument (schwächeres)
  - 3. Gegenargument (schwächstes)
  - Eigene These
  - 4. Argument (mein schwächstes)
  - 5. Argument (besseres)
  - 6. Argument (stärkstes)
- **Zusammenfassung**
  - Meins überwiegt dem anderen

#### Aufbau (Ping-Pong-Prinzip)
- **Einleitung**
  - Interessante Hinführung
  - Deutlich machen, dass es 2 Meinungen gibt
- **Hauptteil**
  - Formulierung beider Thesen
  - 1. Gegenargument (stärkstes)
  - 2. Argument (mein schwächstes)
  - 3. Gegenargument (schwächeres)
  - 4. Argument (besseres)
  - 5. Gegenargument (schwächstes)
  - 6. Argument (stärkstes)
- **Zusammenfassung**
  - Meins überwiegt dem anderen

## Textgebundene Erörterung

#### Vorbereitung
- Ermittle deine Position (These)
- Auf welche Argumente willst du reagieren?
- Welche eigenen Argumente

#### Aufbau
- **Einleitung**
  - Interessante Hinführung
  - Textsorte, Titel, Autor, Erscheinungsdatum
  - Darstellung des Themas
- **Hauptteil**
  - 1. Textanalyse
    - Kurze Inhaltsangabe (Kerngedanke), Darstellung der Intention des Autors der wichtigsten sprachlichen Mittel
  - 2. Erörterung
    - Aufstellung der These und anschließend kritische Stellungnahme zu Hauptargumenten (Zustimmung, Widerspruch oder teilweise Zustimmung) – dabei auch Verwendung eigener Argumente möglich
    - Fazit auf These Bezogen
- **Schluss**
  - Interessanter Ausstieg

#### Tipps
- Im Präsens formulieren und sich von Vorlage sprachlich lösen
- Indirekte Rede für die nicht wörtliche Wiedergabe von Textstellen nutzen (LB S. 296)
- Verwendung korrekter Zitierweise (LB S, 169, 362)

| Zustimmen                                                                                           | Wiedersprechen                                                     | Teilweise zustimmen                                                                 |
| --------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ | ----------------------------------------------------------------------------------- |
| Welche Argumente sind überzeugend und stützen die These des Autors?                                 | Führt der Autor falsche Argumente an?                              | Welche Argumente lassen sich untermauern/entkräften?                                |
| Wie bringt der Verfasser seine Argumente mit der zentralen These in Verbindung?                     | Welche Argumente lassen sich entkräften oder sind nicht schlüssig? | Wie schlüssig sind die zugehörigen Beispiele?                                       |
| Welche zusätzlichen Argumente + Beispiele kannst du anführen, die die These des Autors untermauern? | Sind die Beispiele des Autors nachvollziehbar/plausibel?           | Welche Argumente sind nachvollziehbar/fehlerhaft?                                   |
|                                                                                                     | Gibt es logische Fehlschlüsse?                                     | Welche zusätzlichen Argumente + Beispiele kannst du für beide Positionen aufführen? |
|                                                                                                     | Welche Gegenargumente lassen sich finden?                          |                                                                                     |                                                                                                    |                                                                    |                                                                                     |
