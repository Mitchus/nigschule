# IT-Hardware und Sicherheit

## EVA-Prinzip

```
Eingabe → Verarbeitung → Ausgabe
  (E)        (V)          (A)
```

| Phase          | Geräte                                    |
| -------------- | ----------------------------------------- |
| **Eingabe**    | Tastatur, Maus, Scanner, Mikrofon         |
| **Verarbeitung** | CPU, RAM, Mainboard                     |
| **Ausgabe**    | Monitor, Drucker, Lautsprecher            |
| **Speicherung** | SSD, HDD, USB-Stick, Cloud              |

---

## PC-Komponenten

| Komponente       | Funktion                                    |
| ---------------- | ------------------------------------------- |
| **CPU**          | Zentrale Recheneinheit, führt Befehle aus   |
| **GPU**          | Grafikprozessor, Bildberechnung             |
| **RAM**          | Arbeitsspeicher, flüchtig, schneller Zugriff |
| **Mainboard**    | Verbindet alle Komponenten                  |
| **SSD/HDD**     | Massenspeicher, dauerhaft                   |
| **Netzteil**     | Stromversorgung (ATX-Standard)              |
| **Kühlung**      | CPU-Kühler, Gehäuselüfter                  |

### PC-Vergleich bei Beschaffung

Vergleichskriterien: Leistung, Preis, Energieverbrauch, Erweiterbarkeit, Garantie, Support.

---

## IT-Sicherheit

### Schutzziele (CIA-Triade)

| Schutzziel           | Beschreibung                                |
| -------------------- | ------------------------------------------- |
| **Confidentiality**  | Vertraulichkeit - nur Berechtigte haben Zugriff |
| **Integrity**        | Integrität - Daten sind unverändert         |
| **Availability**     | Verfügbarkeit - Systeme sind erreichbar     |

### Maßnahmen

- **Technisch:** Firewall, Verschlüsselung, Backup, Updates
- **Organisatorisch:** Berechtigungskonzepte, Schulungen, Notfallpläne
- **Physisch:** Zutrittskontrolle, USV, Brandschutz

---

## Green IT & Nachhaltigkeit

Umweltfreundliche Nutzung von IT-Ressourcen:

- **Energieeffizienz:** Sparsame Hardware, Standby-Management
- **Recycling:** Fachgerechte Entsorgung, Kreislaufwirtschaft
- **Virtualisierung:** Weniger physische Server
- **Cloud Computing:** Geteilte Ressourcen, effizientere Auslastung
- **Beschaffung:** Umweltsiegel beachten (Energy Star, Blauer Engel)
