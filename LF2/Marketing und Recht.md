# Marketing und Kaufvertragsrecht

## Marketing-Mix (4 P's)

| P               | Deutsch           | Inhalt                                |
| --------------- | ----------------- | ------------------------------------- |
| **Product**     | Produktpolitik    | Sortiment, Qualität, Design, Service  |
| **Price**       | Preispolitik      | Preise, Rabatte, Zahlungsbedingungen  |
| **Place**       | Distributionspolitik | Vertriebswege, Lagerung, Logistik  |
| **Promotion**   | Kommunikationspolitik | Werbung, PR, Messen, Social Media |

### Marktforschung

| Art             | Beschreibung                                |
| --------------- | ------------------------------------------- |
| **Primärforschung** | Eigene Erhebungen (Umfragen, Interviews) |
| **Sekundärforschung** | Vorhandene Daten nutzen (Statistiken, Berichte) |

### Produktlebenszyklus

```
Umsatz
  ↑
  │        ╱‾‾‾╲
  │      ╱       ╲
  │    ╱           ╲
  │  ╱               ╲
  │╱                   ╲
  └──────────────────────→ Zeit
  Einführung  Wachstum  Reife  Sättigung  Rückgang
```

| Phase          | Umsatz    | Gewinn     | Maßnahmen                |
| -------------- | --------- | ---------- | ------------------------ |
| **Einführung** | Niedrig   | Negativ    | Werbung, Bekanntmachung  |
| **Wachstum**   | Steigend  | Steigend   | Distribution ausbauen    |
| **Reife**      | Maximum   | Maximum    | Differenzierung          |
| **Sättigung**  | Stagniert | Sinkend    | Preissenkung, Varianten  |
| **Rückgang**   | Fallend   | Minimal    | Auslauf oder Relaunch    |

## Kaufvertragsrecht

### Rechtsfähigkeit

| Art                  | Ab wann         | Beispiel                      |
| -------------------- | --------------- | ----------------------------- |
| **Rechtsfähigkeit**  | Ab Geburt       | Erbe empfangen, Eigentümer    |
| **Geschäftsfähigkeit** | Gestaffelt    | Verträge abschließen          |

### Geschäftsfähigkeit

| Stufe                  | Alter      | Umfang                              |
| ---------------------- | ---------- | ----------------------------------- |
| Geschäftsunfähig       | 0-6 Jahre  | Keine gültigen Willenserklärungen   |
| Beschränkt geschäftsfähig | 7-17 Jahre | Mit Einwilligung der Eltern     |
| Voll geschäftsfähig    | Ab 18 Jahre | Alle Rechtsgeschäfte                |

### Willenserklärungen

| Typ              | Beschreibung                                     |
| ---------------- | ------------------------------------------------ |
| **Ausdrücklich** | Mündlich, schriftlich, elektronisch              |
| **Konkludent**   | Durch schlüssiges Handeln (Ware auf Band legen)  |
| **Schweigen**    | Gilt nur bei Kaufleuten als Zustimmung           |

### Zustandekommen eines Kaufvertrags

```
Antrag (Angebot) + Annahme = Kaufvertrag
```

| Element       | Inhalt                                          |
| ------------- | ----------------------------------------------- |
| **Antrag**    | Verbindliches Angebot mit allen wesentlichen Bestandteilen |
| **Annahme**   | Unveränderte Zustimmung zum Antrag              |
| **Pflichten Käufer** | Ware abnehmen, Kaufpreis zahlen          |
| **Pflichten Verkäufer** | Ware liefern, mangelfreie Ware         |

### Leistungsprozess

```
Anfrage → Angebot → Bestellung → Auftragsbestätigung → Lieferung → Rechnung → Zahlung
```
