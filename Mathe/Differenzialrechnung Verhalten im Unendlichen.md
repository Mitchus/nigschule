### Ganz rationale Funktionen

|    | $a>0$ | $a<0$ |
| --- | ----- | ----- |
| $x^{3,5,7}$ | $\lim_{ x \to \pm \infty } f(c)=\pm \infty$ | $\lim_{ x \to \mp \infty } f(c)=\pm \infty$ |
| $x^{2,4,6}$ | $\lim_{ x \to \pm \infty } f(c)=+ \infty$   | $\lim_{ x \to \mp \infty } f(c)=- \infty$   |


### e-Funktionen
|              | $a>0$                                   | $a<0$                                   |
| ------------ | --------------------------------------- | --------------------------------------- |
| $f(x)=a*e^x$ | $\lim_{ x \to + \infty } f(c)=+ \infty$ | $\lim_{ x \to + \infty } f(c)=- \infty$ |
|              | $\lim_{ x \to - \infty } f(c)=0$        |   $\lim_{ x \to - \infty } f(c)=0$                                      |



### Übung

1. $f(x)=3x^4-5x^2$
$\lim_{ x \to \pm \infty } f(c)=+ \infty$

2. $f(x)=-2x^3+x^2-2$
$\lim_{ x \to \mp \infty } f(c)=\pm \infty$

3. $f(x)= \frac{1}{2}x^4+x^3-2x^2+1$
$\lim_{ x \to \pm \infty } f(c)=+ \infty$

4. $f(x)=\frac{5}{2}x^3-x+3$
$\lim_{ x \to \pm \infty } f(c)=\pm \infty$

5. $f(x)=3e^x$
$\lim_{ x \to + \infty } f(c)=+ \infty$
$\lim_{ x \to - \infty } f(c)=0$

6. $f(x)=3(x-1)e^x$
$\lim_{ x \to + \infty } f(c)=+ \infty$
$\lim_{ x \to - \infty } f(c)=0$

7. $f(x)=-\frac{1}{2}(x^4+2)e^x$
$\lim_{ x \to + \infty } f(c)=- \infty$
$\lim_{ x \to - \infty } f(c)=0$
