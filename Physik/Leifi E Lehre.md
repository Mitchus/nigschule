# Elektrisches Feld und Ladungen: Das Wichtigste auf einen Blick

## Elektrisches Feld und Feldliniendarstellung

Das elektrische Feld ($\vec{E}$) um eine Ladung bewirkt die Kraftübertragung auf andere Ladungen im Raum. Die elektrische Feldstärke ($\vec{E}$) ist definiert als:

$$\vec{E} = \frac{\vec{F}}{q_0}$$

wobei $\vec{F}$ die elektrische Kraft auf eine Probeladung $q_0$ ist.

### Punktförmige Ladungen

Für das elektrische Feld $\vec{E}$ um eine punktförmige Ladung $q$ gilt:

$$\vec{E} = \frac{1}{4 \pi \varepsilon_0} \frac{q}{r^2} \hat{r}$$

Hierbei ist $\varepsilon_0$ die elektrische Feldkonstante.

### Zwischen entgegengesetzt geladenen Platten

Für das elektrische Feld $\vec{E}$ im Zwischenraum zweier entgegengesetzt geladener Platten mit Flächeninhalt $A$ und Ladung $Q$ gilt:

$$\vec{E} = \frac{Q}{A \varepsilon_0}$$

## Definition der elektrischen Feldstärke

Die elektrische Feldstärke $\vec{E}$ ist das Maß für die Stärke und Richtung des elektrischen Feldes und wird definiert als:

$$\vec{E} = \frac{\vec{F}}{q}$$

Die Einheit der elektrischen Feldstärke ist Volt pro Meter (V/m).

## Darstellung des elektrischen Feldes

- Das elektrische Feld um eine Ladung verläuft radial zur Ladung.
- Es zeigt von positiven Ladungen weg und zu negativen Ladungen hin.
- Der Betrag des Feldes nimmt mit zunehmendem Abstand zur Ladung ab.

### Elektrisches Feld um eine Punktladung

Die elektrische Feldstärke um eine Punktladung $q$ ist gegeben durch:

$$\vec{E} = \frac{1}{4 \pi \varepsilon_0} \frac{q}{r^2} \hat{r}$$

### Elektrisches Feld zwischen entgegengesetzt geladenen Platten

Das elektrische Feld $\vec{E}$ zwischen entgegengesetzt geladenen Platten ist homogen und konstant:

$$\vec{E} = \frac{Q}{A \varepsilon_0}$$

Hier steht $Q$ für die Ladung auf den Platten und $A$ für die Fläche einer Platte.

## Zusammenfassung

Das elektrische Feld, definiert durch die elektrische Feldstärke $\vec{E}$, ist charakterisiert durch die Ladungen im Raum. Es ist unabhängig von der Probeladung und wird durch die Superposition der Felder verschiedener Ladungen bestimmt.
