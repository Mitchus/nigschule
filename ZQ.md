Im Betrieb arbeite ich hauptsächlich mit Java Spring Boot und React.

### Microsoft Word
Kann ich. Nutze ich für Berichte und Dokumentationen wenn's sein muss. Für technische Docs bevorzuge ich Markdown, ist einfach schneller und flexibler.

Kenne ich gut. Formeln, Pivot-Tabellen, Diagramme - läuft. Nutze ich hauptsächlich wenn Daten schnell visualisiert oder analysiert werden müssen.
